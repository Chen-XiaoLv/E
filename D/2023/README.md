运动强化神经单元，不仅仅是产生大量的血清素和内啡肽，还能分泌神经滋养物质。

睡眠分为四个阶段，前两个阶段是theta波，后两个阶段是delta波，入睡是beta波，清醒是alpha波，前半段快速眼动阶段是在固化记忆，后半段是在修复神经？

睡眠是一定会做梦的，做梦可以把一整天的事情清空。除却睡眠，碳水也可以提高脑细胞活性，提供营养。
